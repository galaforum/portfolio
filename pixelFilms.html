Pixel Films

Role UX Designer and Developer
Team Size Individual

As a part of the application development class, I had to design a website from scratch to let users exchange items. Pixel Films lets users swap movies amongst themselves. Registered

users can update and delete their movies. Users can also accept, reject and withdraw movie swap offers.

Approach

The website was created in a step-by-step manner. I started with designing the structure of the website based on how I wanted to guide the users through the website’s workflow and the professor’s guidelines. Then I designed the website using HTML5, CSS, and NodeJS. I transformed the website into a

Model-View-Controller (MVC) structure, adding all the views and adding routes for the URLs. Creating a MongoDB database and linking everything with database collections dynamized the website interface. Finally, adding user sessions helped to handle user-specific functionalities.
