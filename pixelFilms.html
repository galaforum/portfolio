Pixel Films

Role UX Designer and Developer
Team Size and Duration 1 member, 3 month
Tools Adobe Illustrator, WebStorm

Background

Pixel Films allows users swap movies amongst themselves. Registered users can update and delete their movies. Users can also accept, reject and withdraw movie swap offers. As a part of the application development class, I designed and developed a website from scratch to add a new feature and allow users exchange items.

Approach

The website was created using a user-centered process. I started with designing the structure of the website based on how I wanted to guide the users through the website’s workflow and the professor’s guidelines. Then I developed the website using HTML5, CSS, and NodeJS. I transformed the website into a Model-View-Controller (MVC) structure, adding all the views and adding routes for the URLs. Creating a MongoDB database and linking everything with database collections made the website interface dynamic. Finally, adding user sessions helped to handle user-specific functionality.

Learning Outcomes

1. Design like a developer - As a designer trying to code, I kept the coding constraints in mind, while trying to push technology as far as I could.
